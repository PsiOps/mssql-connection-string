Number Formatter
=========

A small library that adds commas to numbers

## Installation

  `npm install mssql-connection-string`

## Usage

    var numFormatter = require('mssql-connection-string');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`
